## Передача сообщения по UDP, с самостоятельным формированием UDP сегмента, IP пакета и ETH фрейма, используя RAW-сокеты

Программа была запущена на разных компьютерах.

Сначала, с помощью команды `ifconfig` были найдены IP-адрес-а, MAC-адрес-а и название сетевого интерфейса(для клиента)
