## Использование execvp
### Программа выводит температуру и скорость ветра в выбранном городе(Berdsk, Novosibirsk(default), Moscow, St.Petersburg)

## Компиляция
```
make
```

## Работа программы

```
./main
Berdsk                                                      <-------------ВВОД
Weather in Berdsk now: 10°C, wind: moderate 3.4-8.0m/s      <------------ВЫВОД
```
